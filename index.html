PORTFOLIO

MF Webapp

MF is an education foundation dedicated to assisting underprivileged students by facilitating financial aid through donor contributions. Using Java/Kotlin as the backend language and Micronaut as the framework, along with AWS services such as Lambda, S3, API Gateway, and DynamoDB, MF maintains a comprehensive database of student and donor information. The platform ensures transparency by recording detailed records of donations, including the donor, recipient student, and year of contribution. On the frontend, HTML, CSS, and JavaScript are utilized to create an intuitive user interface.

Goddard Webapp

Goddard is a project designed for educational institutions, aimed at simplifying administrative, parental, and billing processes. It employs React.js for frontend and Kotlin/Java, Micronaut, and AWS for backend. Leveraging AWS for scalability and security, it ensures comprehensive data management, facilitating efficient school operations and parental engagement. Its modular architecture and continuous integration enable seamless navigation and rapid feature deployment, enhancing administrative workflows in educational institutions.

Hope3 Application

Introducing the Hope3 Application, an innovative application dedicated to empowering students transitioning from their 12th standard. Developed by a skilled backend team proficient in Java, Spring Boot, and AWS technologies, this platform serves as a vital conduit between students and the Hope3 foundation. Hope3 utilizes Java for backend operations, coupled with the Spring Boot framework and AWS services including EC2, S3, API Gateway, and RDS(MySql). Through this robust architecture, Hope3 maintains a comprehensive database of student and donor information, ensuring transparency and efficiency in financial aid distribution. On the frontend, Hope3 prioritizes user experience, employing HTML, CSS, and JavaScript to craft an intuitive interface for students and donors. Join us in our mission to support students and pave the way for brighter futures with the Hope3 Student Support Platform.
