PORTFOLIO

VarphiKBI

VarphiKBI is a fully-fledged knowledge base integrated model designed to maintain and provide information about specific company proprietary data. Both Admin and Users belonging to that company can browse and retrieve necessary data from the application. Users can also contribute information by uploading files. If an Admin uploads multiple files, they are directly processed and stored inside the knowledge base. However, if a User uploads multiple files, they require Admin review for approval or rejection. The admin is responsible for maintaining the application, tracking inventory, and monitoring application activity via the dashboard.

Goddard

Goddard is a fully-fledged multi-module school management system designed to streamline administrative, parental, and billing processes within educational institutions. It integrates React.js for frontend development with Kotlin/Java, Micronaut, and AWS for backend implementation, offering a seamless platform for managing student and parent data across academic years.

InternalMF

MF is an education foundation dedicated to assisting underprivileged students by facilitating financial aid through donor contributions. Using Java/Kotlin as the backend language and Micronaut as the framework, along with AWS services such as Lambda, S3, API Gateway, and DynamoDB, MF maintains a comprehensive database of student and donor information. The platform ensures transparency by recording detailed records of donations, including the donor, recipient student, and year of contribution. On the frontend, HTML, CSS, and JavaScript are utilized to create an intuitive user interface.
